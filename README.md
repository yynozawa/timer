# 作業タイマー

仕事の種別ごとに作業時間を計測し、日報入力を助けるシンプルな Web アプリです。

## 特徴

- 種別（会社・作業内容）ごとの **複数ストップウォッチ**（同時計測可）
- 各項目を **分:秒** でライブ表示
- 1日の目標分数（既定 **480分**）に対する **割合** を、合計・項目ごとに表示
- 「今日の合計」と日付表示、日付が変われば自動で時間リセット（項目名は維持）
- データはブラウザの **localStorage** に保存（サーバ送信なし）
- 「合計をコピー」で日報用テキストをクリップボードへ
- 「最前面に切り離す」で付箋のように常に最前面の小窓表示（Edge / Chrome の Document Picture-in-Picture）

## 使い方

`index.html` を Edge または Chrome で開くだけです。ビルド不要の静的サイトです。
